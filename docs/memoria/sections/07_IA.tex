\chapter{Uso de Herramientas de Inteligencia Artificial}\label{cap:ia}

\section{Declaración de uso}
Siguiendo las instrucciones de la asignatura, se declara el uso de herramientas de IA generativa para la asistencia en el desarrollo de este trabajo.

\section{Herramientas Utilizadas}
\begin{itemize}
	\item \textbf{ChatGPT/Claude (CBD):} Utilizado para la estructuración de la memoria, redacción de apartados teóricos y depuración de errores en el código Solidity.
	\item \textbf{GitHub Copilot:} Utilizado como asistente de autocompletado de código durante la implementación del frontend en React.
\end{itemize}

\section{Propósito y Metodología}
La IA se ha utilizado para optimizar la redacción y resolver dudas técnicas. Todo el contenido generado ha sido revisado, corregido y validado por los autores, asegurando que el diseño de la arquitectura de datos es propio.
