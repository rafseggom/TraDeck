\chapter*{Abstract}
The Trading Card Game (TCG) market has seen massive growth but continues to face critical challenges, such as the proliferation of counterfeits and a lack of trust in Peer-to-Peer transactions. This project introduces \textbf{TraDeck}, a decentralized application (dApp) designed to ensure authenticity and security in digital asset trading.

The solution leverages Ethereum \textit{Blockchain} technology, utilizing the \textbf{ERC-721} standard for unique card tokenization and the \textbf{ERC-20} standard for managing a stable internal economy. To optimize storage costs, a hybrid persistence architecture has been implemented, delegating metadata and image custody to the \textbf{IPFS} \textit{InterPlanteary File System} decentralized network. The core of the project lies in its smart contracts, which integrate an \textit{Escrow} protocol for secure sales and \textit{Atomic Swaps} that eliminate fraud risk. The result is a functional and accessible platform that demonstrates the feasibility of blockchain as an immutable ledger for modern collecting.
