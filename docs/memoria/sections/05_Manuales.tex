\chapter{Manuales de Usuario y Despliegue}\label{cap:manuales}

\section{Introducción}
Este capítulo proporciona las instrucciones necesarias para desplegar la infraestructura de TraDeck y una guía de uso para el usuario final.

\section{Manual de Despliegue}
\subsection{Requisitos previos}
Instalación de Node.js, Hardhat y configuración de claves API (Pinata, Alchemy/Infura).
\subsection{Despliegue de Smart Contracts}
Uso de \texttt{npx hardhat ignition deploy} para subir los contratos a la red de pruebas Sepolia.
\subsection{Configuración del Frontend}
Configuración del archivo \texttt{.env} con las direcciones de los contratos desplegados.

\section{Manual de Usuario}
\subsection{Conexión de Billetera}
Instrucciones para conectar Metamask y solicitar tokens de prueba (Faucet).
\subsection{Operaciones principales}
Guía visual de cómo crear una carta (minting), ponerla a la venta o proponer un intercambio.

\section{Conclusiones del capítulo}
El sistema está diseñado para ser desplegado de forma sencilla mediante scripts automatizados, facilitando su replicabilidad.
