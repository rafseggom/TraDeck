\chapter{Conclusiones}\label{cap:conclusiones}

\section{Consecución de Objetivos}
Resumen de cómo se han cumplido los objetivos planteados en la introducción: seguridad, descentralización y gestión eficiente de activos TCG.

\section{Limitaciones del Sistema}
Análisis crítico de los problemas encontrados: latencia de la red (Block time), costes de gas y la dependencia de proveedores de IPFS.

\section{Trabajo Futuro}
\begin{itemize}
	\item Implementación de soluciones de Capa 2 (Optimism/Arbitrum) para reducir costes.
	\item Sistema de gobernanza mediante DAOs para la creación de nuevas colecciones.
	\item Integración de oráculos para verificar el valor de las cartas en mercados externos.
\end{itemize}

\section{Valoración Personal}
Reflexión sobre el aprendizaje en la gestión de bases de datos descentralizadas y el desarrollo Web3.
